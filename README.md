## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/priyanshu-samal-portfolio.git
```

2. Install dependencies:
```bash
cd priyanshu-samal-portfolio
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
